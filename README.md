# Reversible-chain-scission
This repository shares codes and data created based on the breakable freely jointed chain model by Ansari, Liechti, Makarov  and Huang (2026) to predict the statistics and kinetics of polymer chain scission and healing. 

Matlab scripts:

barriers.m - calculated energy barriers for 5 cases

rate_integrated.m - integrating the rate equation for a constant pulling rate

KMC1.m - kinetic Monte Carlo simulation at a constant end-to-end distance
